# photo_mover

Python CLI utility to move photos and videos from a source tree to a destination directory.

Quick start

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m photo_mover --src "C:\path\to\source" --dst "D:\path\to\dest" --dry-run
```

See `--help` for options.
